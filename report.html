Pseudo Code :

Input : N,P1,P2,...,PN,C

Segmented-Least-Squares() {

  M[0] = 0

  for j = 1 to N

    for i = 1 to j

      compute the least square error eijfor the segment pi,..., pj

  for j = 1 to N

    M[j] = min1<=i<=j (eij+ C + M[i-1])

  return M[N]

}

Running Time :

O(n3)↙can be improved to O(n2) by pre-computing various statistics

Observations :

  • Number of Line segments decrease with increase of value of C

  • After certain value of C,it behaves as a best-fit line

  • At C = 0,Error is 0

Limitations :

  • It may not give correct output if the input contains two points with same x-coordinate
